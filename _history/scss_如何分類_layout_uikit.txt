[ 不獨立成一工具組 , 僅是片段 selector 者用 .layout- ]
[ 如同 bootstrap 片段 unities 者為此 ]
路徑 : scss/layout/_common.scss (原則上彙整入此)
prefix (前綴詞) .layout-
ex:
.layout-shadow




[ 多個 selector 組合成一工具組者 ]
路徑 : scss/uikit/_每個工具組.scss
prefix : .uikit-工具名- (只有自創的元件能這麼做)
命名:
自創 > _uikit-工具.scss ( x_kit-工具.scssx )
模組 > _tmp-工具.scss
